NIKONEKT APP USER GUIDE

We have created convenient and easy way for customers to interact with TANESCO through the Nikonekt App. Kindly follow the below procedure to install and use it:

Installation
  1. Search the word Nikonek from play store or Apple store
  2. Click Install
  3. Click Open
  4. See available services
Nikonekt App offers the following services
  1. Online Power Application
  2. Emergency/Fault
  3. Enquires
  4. Complaints
  5. Request
  6. Check status
  7. List of eligible electrical installation personnel
  8. Information
Nikonekt App settings

To make life much better for Nikonekt App users the following settings can be done

  1. Language switching
    2.
    1. Open Nikonekt App
    2. Click setting
    3. Click language
    4. Select English or Kiswahili to switch language
    5. System will accommodate language of your choice.
  2. Updating Meter Number
    3.
    1. Open Nikonekt App
    2. Click setting
    3. Select Customer meter
    4. Enter your meter number
    5. Click Verify to check existence of the meter
    6. Click okay go back to the setting menu
  3. Updating Mobile number
    4.
    1. Open Nikonekt App
    2. Click setting
    3. Select Mobile
    4. Enter your mobile number
    5. Click update
  4. Updating Location details
    5.
    1. Open Nikonekt App
    2. Click setting
    3. Click location
    4. Select Region, District, Area, Village/Street
    5. Enter special mark (famous person, well known office, or any other mark that will help TANESCO to reach your premise)
How to use the Nikonekt App?
  1. Power Application
    2.
    1. Open Nikonekt App
    2. Click New Application
    3. Select Application purpose
    4. Select Application category
    5. Select Application type
    6. Select identification type (NIDA No for individuals and TIN No for institution/company)
    7. Click start Application to continue
    8. Enter Applicant profile details
    9. Enter Contacts person details
    10. Enter property details
    11. Select electrical system in the building
    12. Click Submit Application to continue
    13. Read and agree with terms and conditions of power supply
    14. Click complete applications to submit an application.
    15. Share system generated application number and phone number to the contractor of your choice to enable him to fill in wiring details and submit the same to TANESCO.
  2. Check Application status
    3.
    1. Open Nikonekt App
    2. Click status (Check application status)
    3. Click check status
    4. View current application status
    5. Click okay to go back
  3. View list of eligible electrical contractors
    4.
    1. Open Nikonekt App
    2. Click contractors
    3. Scroll up to see more contractors
    4. Click contractor of your choice to see more details
  4. Reporting emergency/Fault
    5.
    1. Open Nikonekt App
    2. Select Emergency
    3. Enter Meter Number
    4. Click Verify to validate the meter
    5. Enter Contact details
    6. Enter problem details
    7. Enter Location descriptions
    8. Click submit emergency
    9. View ticket number for your reference
  5. Make an enquire
    6.
    1. Open Nikonekt App
    2. Select enquiry
    3. Enter Meter Number
    4. Click Verify to validate the meter
    5. Enter Mobile number
    6. Select enquiry type
    7. Enter enquiry details
    8. Click submit enquiry
    9. View enquire ticket number for your reference.
  6. File a complaint
    7.
    1. Open Nikonekt App
    2. Select Complaint
    3. Enter Meter Number
    4. Click Verify to validate the meter
    5. Enter Mobile number
    6. Click choose to pick the previous ticket number youâ€™re complaining about.
    7. Click pick to select the previous ticket
    8. Select complaint type
    9. Enter complaint details
    10. Click submit complaint
    11. View complaint ticket number for your reference.
  7. Make Request
    8.
    1. Open Nikonekt App
    2. Select request
    3. Enter Meter Number
    4. Click Verify to validate the meter
    5. Enter Mobile number
    6. Select request type
    7. Enter request details
    8. View request ticket number for your reference
  8. Check Emergency, complaint, enquiry and request status
    9.
    1. Open Nikonekt App
    2. Select respective service
    3. Click three dots at the right of your ticket number
    4. Click check status to view the current progress status
    5. You may click delete to remove respective ticket from your history
